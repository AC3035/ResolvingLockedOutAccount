<h1>Resolving a locked out User Account.</h1>


<h2>Description</h2>
This project was used to simulate a real world scenario where a user account get locked out due to failed login attempts.
<br />


<h2>Technologies Used </h2>

- <b>Windows 11 Client</b> 
- <b>Windows Server 2022</b> 
- <b>Active Directory Domain Services</b>
